# /artifex reference library

Curated montages from the ref-study of 12 award-caliber sites (real fonts read from each DOM, captured headless). These ship with the skill so `/artifex` designs **grounded in studied technique, not the model's idea of "engaging" (N8)**. Study the montage for the technique you're about to build before you build it.

**Full set** (all 12 sites, per-site screenshots, `_analysis-notes.md`, the original variance-playbook): `~/claude/notes/pulse-pitch-deck-2026-06-20/ref-study/`. The six bundled here are the most instructive + reasonably sized.
